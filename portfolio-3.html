HostHaven – Accommodation Booking Platform

hosthaven-w6l1.onrender.com

A full-stack Airbnb-inspired booking platform with geospatial search, secure payments, and real-time availability management.

Complete property management system enabling hosts to create detailed listings with multi-image uploads, pricing calendars, amenity selection, and availability blocking.

Advanced geospatial search powered by Mapbox GL JS with interactive map clustering, custom markers, radius-based filtering, and location autocomplete using geocoding APIs.

Intelligent booking engine with conflict detection, instant booking and request-to-book options, calendar synchronization, and automated email confirmations to both hosts and guests.

Secure authentication flow using Passport.js supporting local strategy with bcrypt password hashing, session management with express-session, and OAuth integration for social logins.

Key Features

  • Multi-image Upload with Cloudinary
  • Interactive Map Search
  • Real-time Availability Calendar
  • Server-Side Rendered Views (EJS)
  • OAuth Social Login
  • CDN Image Optimization
