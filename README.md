# Skydome Aviation - IT & Tech Division Website

Welcome to the repository for the official website of the **IT & Tech Division of Skydome Aviation**. This platform showcases our custom enterprise web applications, educational dashboards, learning management systems, and specialized aviation software integrations.

## Features
- **Aviation-grade IT Solutions**: Details about our enterprise ERP and Learning Management System (LMS) development.
- **Project Showcases**: Features real-world platforms built by our division, including:
  - *Skydome Aviation Main Website*
  - *Aviation CET Official Portal*
  - *HostHaven Accommodation Booking Engine*
- **User-Centric Design**: Modern UI/UX layout optimized for both light and dark themes.
- **Interactive Forms**: Secure, direct customer contact form integrated with automated backend notifications.

## Technologies Used
- HTML5, CSS3, & Vanilla JavaScript
- Bootstrap (v5.3.6) & Bootstrap Icons
- Typed.js (Typing animation library)
- Swiper (Touch slider integration)
- AOS (Animate On Scroll library)
- Web3Forms API (Secure form handling)
